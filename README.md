# MR CyberSword Telegram Bot

- Polling-based Telegram Bot
- Force Join Group + Nu
- mber info API
- Deploy anywhere PHP 7.4+ supported
- 24/7 run: Render.com / Railway / Fly.io recommended

## Usage

1. Set your `BOT_TOKEN` in `bot.php`
2. Set your `FORCE_GROUP_ID` and `GROUP_LINK`
3. Deploy to Render / Railway
4. Bot will run 24/7, `/start` and number info ready
